Paging.js使用方法

  1. 下载Paging.js插件(点此下载)，该插件依赖JQuery库(点此下载)
  2. 在<head>标签中导入JQuery库和Paging.js插件: 
    <script src="js/jquery-1.11.3.min.js" type="text/javascript" charset="utf-8"></script>
<script src="js/page4.js" type="text/javascript" charset="utf-8"></script>
  3. 调用方法：$(selection).paging(object);
  4. object参数列表：
    • pageCount（必需）：页码总数
    • initPage：初始页码位置
    • showCount：每次显示多少页
    • fn：回调函数，参数为当前页码数
    • liCss：默认页码样式，JSON格式
    • liActive：选中时页码的样式，JSON格式
    • liForbid：不可用时页码的样式，JSON格式

Demo

$('#div2').paging({
'pageCount': 20,
'initPage': 1,
'showCount': 5,
'fn': function a(pageNum) {
	alert('页码为：' + pageNum);
},
'liCss': {
'display': 'inline-block',
'border': '#8db5d7 1px solid',
'padding': '2px 8px',
'cursor': 'pointer',
'margin': '0 4px',
'color': '#000',
'border-radius': '5px',
'pointer-events': 'auto'
},
'liActive': {
'border': '#e89954 1px solid',
'font-weight': 'bold',
'background': '#ffca7d',
'cursor': 'default',
'pointer-events': 'none'
},
'liForbid': {
'border': '#ccc 1px solid',
'color': '#ccc',
'cursor': 'default',
'pointer-events': 'none'
}});
